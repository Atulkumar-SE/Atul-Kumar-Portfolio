# Portfolio Website

A sleek, premium, high-performance developer portfolio built with modern frontend architecture. This project showcases interactive UI/UX designs, responsive web development layouts, and structured project modules with a focus on seamless user experiences.

## 🚀 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (App Router architecture)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for fluid, utility-first configurations
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) for fluid page transitions, interactive states, and infinite keyframe loops
*   **Typography:** [Google Fonts (Syne)](https://fonts.google.com/) for geometric branding layouts

---

## 🛠️ Key Project Architecture

The current project structure is organized cleanly to maintain a modular workflow:

*   **`src/app/`**: Contains the main Next.js App Router layout, page configuration, and styling frameworks.
*   **`public/`**: Stores static visual assets, vector logos, and global media files.
*   **Tailwind Configuration**: Pre-configured custom theme scales (such as dedicated brand accent colors like `#FF9800` and typography mapping).
